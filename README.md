# Catalogo
¡Bienvenido! Este espacio está dedicado a la recopilación y distribución de archivos stl y 3mf. Aquí encontrarás todo el material organizado y listo para descargar.

## 📥 ¿Cómo descargar los archivos?

Tienes dos opciones principales para obtener tu material:

1. **Descargar archivos individuales:** Haz clic sobre el archivo que deseas descargar y luego presiona el botón **"Download"** (o el ícono de descarga) en la esquina superior derecha de la pantalla.
2. **Descargar toda la carpeta:** Ve a la página principal de este repositorio, haz clic en el botón verde **"Code"** y selecciona **"Download ZIP"** [Descarga de archivos de código fuente](https://docs.github.com/es/repositories/working-with-files/using-files/downloading-source-code-archives).
